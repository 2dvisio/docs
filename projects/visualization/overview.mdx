# Visualization projects

This section showcases interactive web-based visualization tools demonstrating applications in health monitoring, mathematics, and computer graphics.

## Projects

### Respiratory rate monitor
Real-time respiratory rate detection using webcam and signal processing algorithms. This tool demonstrates computer vision applications in health monitoring by analyzing subtle motion patterns to estimate breathing rate.

**Key features:**
- Webcam-based motion detection
- Signal processing for respiratory pattern extraction
- Real-time visualization of breathing waveforms
- Non-contact monitoring approach

[Learn more about respiratory rate monitoring →](/projects/visualization/respiratory-rate)

### Fractals and mathematical curves
Interactive visualizations of mathematical concepts including:
- Fractal patterns (Mandelbrot set, Julia sets)
- Space-filling curves (Hilbert curve, Peano curve)
- Geometric transformations
- Recursive algorithms

**Key features:**
- Interactive parameter controls
- Real-time rendering
- Educational demonstrations
- Exploration of mathematical beauty

[Explore fractals and curves →](/projects/visualization/fractals)

## Technical approach

These projects use web technologies to create accessible, interactive visualizations:
- HTML5 Canvas for rendering
- JavaScript for computation and interaction
- Real-time processing and animation
- Responsive design for various devices

## Purpose

These visualization tools serve multiple purposes:
- **Education**: Demonstrate mathematical and computational concepts
- **Research**: Prototype algorithms for health monitoring
- **Exploration**: Enable interactive discovery of patterns and behaviors
- **Accessibility**: Make complex concepts tangible through visualization

## Try them out

All visualization tools are web-based and can be accessed directly through your browser. No installation or special software required.
